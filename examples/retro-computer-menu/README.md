# Retro Computer Menu Demo

This example now includes a playable **Tic-Tac-Toe** (with a sarcastic CPU at configurable intelligence) and **Chess** (with Easy/Normal/Hard bot difficulties and reactive trash talk).

## Run in your browser

From the repository root:

```bash
npm run retro-menu
```

Then open:

- http://localhost:4173/

This URL is served from this project at `examples/retro-computer-menu`.
